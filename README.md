## 特性

- **低代码开发，** 快速生成原代码。
- **可视化开发，** 通过GUI生成页面。
- **物料市场，** 代码资源共享，包含组件、区块、动态容器、场景。

## 快速使用
```bash
# 全局安装
$ npm install -g sparrow-code

# 运行
$ sparrow
```
